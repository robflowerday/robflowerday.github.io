Machine Learning with Python

The aim of this course is to teach Machine Learning to a
relative begginer to aid them in becoming proficient in
building and understanding very useful Machine Learning
models.

Pre-requisites of this course are being able to work
with a few python libraries: pandas, numpy and
matplotlib, and a basic knoledge of statistics would
be useful.

I add new material to this course frequently and so
you may notice changes as you visit the site.

If you have any questions or feedback, feel free to
message me on rob.flowerday@hotmail.co.uk.

Lesson 1: Types of Machine Learning

Gain an understanding of the 3 main types of Machine Learning problems and Models, Supervised Learning, Unsupervised Learning and Reinforcement Learning.

Lesson 2: K-Nearest Neighbors

Start your journey and train your first model by importing your first dataset from the sklearn library, visualising the data and gaining an intuition fo how a K-Nearest Neighbors model works and learning some code.
